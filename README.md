# Shorti
Платформа просмотра коротких видео. Микросервисная реализация с Ef Core, AutoMapper, Ocelot, Elastic Search, IdentityServer, автодокументация Swagger. Клиент на React.

## Сервисы:
* Identity
* Video
* Activities
